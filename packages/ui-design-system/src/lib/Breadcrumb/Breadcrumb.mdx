`Breadcrumb` renders an accessible `<nav>` with an ordered list of crumbs. The last item is the current page (rendered as `<span aria-current="page">`). Earlier crumbs render as `<a>` links when `href` is provided, or as `<button>` elements when only `onClick` is provided (for SPA navigation).

## Usage

```tsx
import { Breadcrumb } from 'ui-design-system';

<Breadcrumb
  items={[
    { label: 'Home',     href: '/' },
    { label: 'Settings', href: '/settings' },
    { label: 'Profile' },
  ]}
/>
```
