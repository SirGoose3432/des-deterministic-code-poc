`Alert` surfaces contextual feedback to the user. It supports four semantic variants — info, success, warning, and error — each with distinct colour and icon treatment. Alerts can optionally include a title and a dismiss button.

## Usage

```tsx
import { Alert } from 'ui-design-system';

<Alert variant="success" title="Saved">
  Your changes have been saved.
</Alert>
```
