`Textarea` is a controlled multiline text input with a label, optional character counter, error state, and configurable resize behaviour.

## Usage

```tsx
import { Textarea } from 'ui-design-system';

<Textarea
  id="bio"
  label="Bio"
  value={bio}
  onChange={setBio}
  placeholder="Tell us about yourself…"
  maxLength={500}
/>
```
