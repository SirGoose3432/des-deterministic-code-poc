`Button` is the base action primitive for the design system. It supports four visual variants, three sizes, loading and disabled states, and full-width layout.

## Usage

```tsx
import { Button } from 'ui-design-system';

<Button label="Save changes" variant="primary" onClick={handleSave} />
```

### Primary

The default variant for primary actions.
