`Spinner` is a standalone loading indicator with three sizes and an accessible `role="status"` label. Use it for async states that aren't tied to a specific button (prefer the `loading` prop on `Button` when the async action is button-triggered).

## Usage

```tsx
import { Spinner } from 'ui-design-system';

<Spinner size="md" label="Loading data…" />
```
