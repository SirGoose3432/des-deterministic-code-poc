`TextInput` is a controlled form field with a visible label, optional hint text, and inline validation error display. It is fully accessible: the label is wired via `htmlFor`, errors use `role="alert"`, and the input carries `aria-invalid` and `aria-describedby` when appropriate.

## Usage

```tsx
import { TextInput } from 'ui-design-system';

const [value, setValue] = useState('');

<TextInput
  id="email"
  label="Email address"
  value={value}
  onChange={setValue}
  type="email"
  placeholder="you@example.com"
/>
```

> `TextInput` is a **controlled component** — you must supply both `value` and `onChange`.
