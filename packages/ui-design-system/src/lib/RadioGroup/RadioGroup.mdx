`RadioGroup` is a controlled group of radio inputs wrapped in a `<fieldset>` for accessibility. Supports vertical and horizontal layouts, error state, and per-option disabled flags.

## Usage

```tsx
import { RadioGroup } from 'ui-design-system';

<RadioGroup
  name="plan"
  legend="Billing plan"
  value={plan}
  onChange={setPlan}
  options={[
    { value: 'free', label: 'Free' },
    { value: 'pro',  label: 'Pro' },
  ]}
/>
```
