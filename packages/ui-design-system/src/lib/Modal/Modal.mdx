`Modal` is a portal-based overlay panel. It renders into `document.body` via `createPortal`, locks body scroll while open, traps focus in the panel, and dismisses on Escape key or backdrop click.

## Usage

```tsx
import { Modal } from 'ui-design-system';

const [open, setOpen] = useState(false);

<Modal
  isOpen={open}
  onClose={() => setOpen(false)}
  title="Confirm action"
>
  Are you sure you want to proceed? This action cannot be undone.
</Modal>
```

## Accessibility

- The panel carries `role="dialog"` and `aria-modal="true"`.
- The title is linked via `aria-labelledby`.
- Focus moves into the panel on open via `tabIndex={-1}` and `.focus()`.
- Pressing **Escape** always calls `onClose`, regardless of the `closeOnBackdropClick` setting.
