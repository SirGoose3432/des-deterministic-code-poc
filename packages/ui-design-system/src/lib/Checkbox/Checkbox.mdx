`Checkbox` is a controlled boolean input with a visible label, optional hint, error state, and indeterminate support.

## Usage

```tsx
import { Checkbox } from 'ui-design-system';

<Checkbox
  id="terms"
  label="I agree to the terms"
  checked={accepted}
  onChange={(e) => setAccepted(e.target.checked)}
/>
```
