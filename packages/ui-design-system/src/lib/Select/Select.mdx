`Select` is a controlled dropdown field with a label, placeholder, error, and hint. It wraps a native `<select>` for accessibility and keyboard navigation.

## Usage

```tsx
import { Select } from 'ui-design-system';

<Select
  id="department"
  label="Department"
  value={value}
  onChange={(e) => setValue(e.target.value)}
  options={[
    { value: 'design', label: 'Design' },
    { value: 'engineering', label: 'Engineering' },
  ]}
  placeholder="Select a department…"
/>
```
