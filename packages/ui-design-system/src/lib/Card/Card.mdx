`Card` is the standard content container. It provides a white surface with a border, optional header title, configurable padding, and shadow elevation.

## Usage

```tsx
import { Card } from 'ui-design-system';

<Card title="User profile" padding="md" elevation="sm">
  Content goes here.
</Card>
```
